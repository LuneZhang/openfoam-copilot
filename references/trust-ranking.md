# Trust Ranking

## High Trust
- official OpenFOAM user guides
- official OpenFOAM solver and utility documentation
- official OpenFOAM tutorials
- official repository documentation and authoritative issue discussions

## Medium Trust
- experienced community troubleshooting threads with concrete reproduction details
- repeated guidance corroborated by multiple high-quality users or official docs
- community explanations that align with official semantics but add practical debugging order

## Low Trust
- single-author advice without validation
- tips that solve one case but have unclear generality
- folklore-style recommendations without scope or explanation

## Usage Rules
- high-trust content can inform default guidance
- medium-trust content should be framed as practical heuristics
- low-trust content must be clearly labeled and never used as canonical setup rules without corroboration
