# Prompt Layer

These prompts define how an agent should actually use the project.

Current prompts:
- `setup-assistant.md`
- `troubleshooting-assistant.md`
- `case-review.md`
- `source-triage-assistant.md`

Recommended usage order:
- setup task -> `setup-assistant.md`
- debugging task -> `troubleshooting-assistant.md`
- auditing an existing case -> `case-review.md`
- source ingestion / curation -> `source-triage-assistant.md`
